# API Documentation

Base URL (local): `http://127.0.0.1:8000`

Interactive docs are also auto-generated by FastAPI at `/docs` (Swagger) and `/redoc`.

All responses are JSON. Successful responses include `"success": true`.

---

## 1. POST `/summarize`

Summarizes a block of text.

### Request Body
| Field      | Type | Required | Constraints                  | Description                          |
|------------|------|----------|-------------------------------|---------------------------------------|
| text       | string | Yes    | 20–20000 characters           | The text to summarize                |
| max_words  | int    | No     | 10–500 (default: 100)         | Approx. max length of the summary    |

### Example Request
```json
POST /summarize
{
  "text": "FastAPI is a modern, fast web framework for building APIs with Python based on standard type hints. It is one of the fastest Python frameworks available, on par with NodeJS and Go, thanks to Starlette and Pydantic.",
  "max_words": 30
}
```

### Example Response (200)
```json
{
  "success": true,
  "summary": "FastAPI is a fast, modern Python web framework for building APIs, using type hints, and performing on par with NodeJS and Go."
}
```

### Validation Error Example (422)
```json
{
  "detail": [
    {
      "type": "string_too_short",
      "loc": ["body", "text"],
      "msg": "String should have at least 20 characters",
      "input": "short"
    }
  ]
}
```

## 2. POST `/translate`

Translates text into a target language.

### Request Body
| Field            | Type   | Required | Constraints        | Description                          |
|------------------|--------|----------|---------------------|----------------------------------------|
| text             | string | Yes      | 1–5000 characters   | The text to translate                |
| target_language  | string | Yes      | 2–50 characters     | e.g. `"French"`, `"Hindi"`, `"es"`   |

### Example Request
```json
POST /translate
{
  "text": "Good morning, how are you?",
  "target_language": "French"
}
```

### Example Response (200)
```json
{
  "success": true,
  "translated_text": "Bonjour, comment allez-vous ?",
  "target_language": "French"
}
```

## 3. POST `/generate-email`

Generates a draft email (subject + body) for a given purpose and tone.

### Request Body
| Field           | Type   | Required | Constraints       | Description                                     |
|-----------------|--------|----------|---------------------|--------------------------------------------------|
| purpose         | string | Yes      | 5–2000 characters  | What the email is about                         |
| tone            | string | No       | default: "professional" | e.g. "friendly", "formal", "apologetic"   |
| recipient_name  | string | No       | max 100 characters | Optional, used to personalize the email         |

### Example Request
```json
POST /generate-email
{
  "purpose": "Follow up after a job interview for the AI/ML Intern role",
  "tone": "professional",
  "recipient_name": "Ms. Sharma"
}
```

### Example Response (200)
```json
{
  "success": true,
  "subject": "Following Up on AI/ML Intern Interview",
  "body": "Dear Ms. Sharma,\n\nThank you for taking the time to speak with me about the AI/ML Intern position. I enjoyed our conversation and remain very enthusiastic about the opportunity to join the team...\n\nBest regards"
}
```

## Error Responses (apply to all endpoints)

### 422 — Validation Error
Returned automatically by FastAPI/Pydantic when a request body fails validation (missing field, too short, wrong type, etc).

### 502 — AI Service Error
Returned when the Gemini API call fails (bad key, network issue, empty response).
```json
{
  "success": false,
  "error": "AI_SERVICE_ERROR",
  "message": "Failed to get a response from the AI provider: <details>"
}
```

### 500 — Internal Server Error
Returned for any unexpected/unhandled exception.
```json
{
  "success": false,
  "error": "INTERNAL_SERVER_ERROR",
  "message": "Something went wrong. Please try again later."
}
```
